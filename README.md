# おでかけ天気サポート

## 概要

雨が降りそうな日に、傘や上着が必要かを確認できるWebアプリです。
地域を選択すると、天気APIから当日の気温・降水確率を取得して表示します。

## 現在動く機能

* [x] 日本国内の地域を選択できる
* [x] 選択した地域を `settings.json` に保存できる
* [x] Open-Meteo APIから当日の天気情報を取得できる
* [x] 現在の気温と最大降水確率を表示できる
* [x] 時間別の気温と降水確率を表示できる
* [x] 降水確率に応じて傘の必要性を表示できる
* [x] 気温に応じて上着のおすすめを表示できる
* [x] 天気情報を取得できない場合にエラーメッセージを表示できる

## 使用技術

* Python
* Flask
* HTML / CSS
* Open-Meteo Weather API
* JSONファイルによる地域設定の保存

## 起動方法

### 1. 仮想環境を有効化する

```powershell
.\.venv\Scripts\Activate.ps1
```

### 2. 必要なライブラリをインストールする

```powershell
pip install -r requirements.txt
```

### 3. アプリを起動する

```powershell
python app.py
```

### 4. ブラウザで開く

```text
http://127.0.0.1:5000
```

## 動作確認手順

1. ブラウザで `http://127.0.0.1:5000` を開く
2. 地域を東京・大阪・札幌・福岡・那覇から選ぶ
3. 「更新」ボタンを押す
4. 選択地域の天気、気温、降水確率が表示されることを確認する
5. 傘・上着のおすすめが表示されることを確認する
6. ページを更新しても選択地域が維持されていることを確認する
